
# go-elasticsearch

Sample project to explore Golang integration with Elasticsearch

---

## Run

1. Start Elasticsearch

2. Run sample

    - `olivere/elastic` sample
    
        ```shell script
        cd olivere-elastic
        go run main
        ```

    - `elastic/go-elasticsearch` sample
    
        ```shell script
        cd go-elastic
        go run main
        ```
